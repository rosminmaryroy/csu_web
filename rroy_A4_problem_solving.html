Why Problem Solving Matters at Verdant Web Solutions

Every website launch hits an unexpected snag: a hosting quirk, a broken plugin, a client who changes their mind two days before go-live. A developer who can break a problem into pieces and ship a fix saves the team time and the client stress.

Examples of My Problem Solving

Python Assessment Refactor

  1. Identified that my Assessment 4 Python file had repeated input-validation logic
  2. Extracted the logic into a single helper that returns a typed result
  3. Cut the file size by roughly 30% and made the test cases far easier to read

Spice Blend Booking Conflict

At Spice Blend Restaurant, a double-booking issue was costing the team time every weekend. I sketched the booking flow on paper, found the gap (no confirmation step between phone and online bookings), and proposed a shared spreadsheet checked at the start of every shift. Double bookings dropped to near zero within a month.

My Approach

  • Reproduce the issue first — don't trust the bug report alone
  • Write down what I know and what I'm assuming
  • Try the smallest, cheapest fix that could plausibly work
  • Document the fix so the next person doesn't repeat the chase

Motto: find the root cause, not just a workaround.
